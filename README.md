# psicologa-arianne-fontes

Landing Page - Psicologia Clínica (Dra. Arianne Fontes)

Sobre o Projeto:
Desenvolvimento de uma Landing Page responsiva e elegante voltada para a área da saúde (Psicologia Clínica). O objetivo do projeto foi criar uma interface acolhedora, que transmitisse profissionalismo e facilitasse a conversão (agendamento de consultas). O projeto conta com funcionalidades interativas que alternam layouts e visibilidade de sessões sem a necessidade de recarregar a página.

Funcionalidades Principais:
•	Design Responsivo & Mobile-First: Layout fluido utilizando Grid System do Bootstrap 5, adaptável a todos os dispositivos.
•	Sessão de Especialidades Dinâmica: Implementação de lógica em JavaScript para alterar a estrutura do grid em tempo real (de 2 colunas com sidebar para 3 colunas full-width) ao expandir o conteúdo.
•	Navegação SPA (Single Page Application) Simulada: Alternância entre a "Landing Page" e a "Área do Blog" controlada via JavaScript, mantendo a performance instantânea.
•	Componentes Interativos: FAQ no estilo acordeão, botões de Call-to-Action (CTA) estratégicos e botão flutuante do WhatsApp.
•	Estilização Personalizada: Uso de variáveis CSS para paleta de cores (Verde Petróleo/Dourado) e tipografia elegante (Playfair Display e Lato).


Tecnologias Utilizadas:
•	HTML5 Semântico
•	CSS3 (Customizado + Animações Keyframes)
•	Framework Bootstrap 5.3
•	JavaScript (Vanilla/ES6) para manipulação do DOM e eventos.
